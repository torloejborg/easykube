== Methods

=== sparseCheckout

Downloads a particular directory in a git repository

[source,javascript]
----
sparseCheckout(repository, branch, directories, destination)
----

.Parameters
[cols="1,1,3"]
|===
| Name | Type | Description

| `repository`
| string
| the git repository
| `branch`
| string
| which branch to check out
| `directories`
| Array.&lt;string&gt;
| which directories in the repository to check out sparsely
| `destination`
| string
| directory for checkout

|===
