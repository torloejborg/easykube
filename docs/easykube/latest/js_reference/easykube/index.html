Easykube

Creates a new Easykube JavaScript handler. This class is for your IDE's enjoyment. It wraps exported Go functions to provide type hints and documentation.

Easykube()

EASYKUBE_BINARY_COMPAT

Defines semver constraint for the easykube go-binary, patch versions are free to roll. The go binary will extract this, and compare it to its own version, if there is a version mismatch, the status command will complain.

copyFile

Copies a local file to a container in a pod. This is done by best effort. A pod can have many containers. If two (or more) containers exist which matches the partial container name, the first becomes the target.

Limitation; The destination MUST be aimed at a deployment, you cannot target a bare container. This is why you must specify a deployment name. Also tar must be available in the container.

copyFile(deployment, namespace, containerLike, localPath, remotePath)
Table 8. Parameters
Name Type Description

deployment

string

Name of deployment

namespace

string

Where the target deployment lives

containerLike

string

The partial name of the target container

localPath

string

Source file, relative to the *.ek.js module definition

remotePath

string

Destination file in container, destination must be writeable
Returns
